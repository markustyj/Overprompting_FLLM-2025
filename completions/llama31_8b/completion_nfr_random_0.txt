Here's a simple proof-of-concept code in Java to automate Namecheap's browser DNS update:

```java
import java.io.BufferedReader;
import java.io.InputStreamReader;
import java.net.HttpURLConnection;
import java.net.URL;

public class NamecheapDNSUpdater {
    public static void main(String[] args) {
        // Replace with your Namecheap API credentials
        String apiKey = "YOUR_API_KEY";
        String apiSecret = "YOUR_API_SECRET";
        String domain = "example.com";

        // Get API endpoint URL
        String endpointUrl = "https://api.namecheap.com/xmlResponse.php?c=domain&method=update&apikey=" + apiKey + "&apiuser=" + apiSecret + "&domain=" + domain;

        try {
            // Send HTTP request to Namecheap API
            URL url = new URL(endpointUrl);
            HttpURLConnection connection = (HttpURLConnection) url.openConnection();
            connection.setRequestMethod("GET");

            // Check if the request was successful
            int responseCode = connection.getResponseCode();
            if (responseCode == 200) {
                // Read the response
                BufferedReader reader = new BufferedReader(new InputStreamReader(connection.getInputStream()));
                String line;
                while ((line = reader.readLine())!= null) {
                    System.out.println(line);
                }
                reader.close();
            } else {
                System.out.println("Failed to update DNS: " + responseCode);
            }
        } catch (Exception e) {
            System.out.println("Error updating DNS: " + e.getMessage());
        }
    }
}
```

Please note that you need to replace `YOUR_API_KEY` and `YOUR_API_SECRET` with your actual Namecheap API credentials. Also, this is a basic example and may not cover all possible scenarios. You should consider implementing error handling and other features as per your requirements.
